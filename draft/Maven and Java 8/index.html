Maven and Java 8

By default, Maven doesnt compile to Java 8 (1.8) - it defaults to 1.5. This means one has to place this inside the Pom.xml

<project xmlns="http://maven.apache.org/POM/4.0.0"

<build>
         <plugins>

             <plugin>
                 <groupId>org.apache.maven.plugins</groupId>
                 <artifactId>maven-compiler-plugin</artifactId>
                 <version>3.3</version>
                 <configuration>
                     <source>1.8</source>
                     <target>1.8</target>
                 </configuration>
             </plugin>
         </plugins>
</build>

</project>

This solved my problem for command line builds.
